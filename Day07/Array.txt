배열 = 같은 데이터 타입을 가지는 서로 다른 값들이
          하나의 변수로 처리되는 자료 구조

선언 - 자료형[] 배열이름;


생성 - 배열이름 = new 자료형[길이];

선언, 생성
 - 자료형[] 배열이름 = new 자료형[길이];

배열의 초기화
 - 배열은 생성과 동시에 자신의 타입에 해당하는 기본값으로
   초기화된다.
 - 생성된 배열의 각 저장 공간은 
   배열이름[인덱스]
- 인덱스의 범위 : 0~배열길이-1
